What happens next

Your values are sent to a LangGraph multi-agent system that investigates before producing a result. Every run is traced in LangSmith.

Investigator agent tools

  • run_prediction (RF model)
  • get_shap_values (per-prediction)
  • get_feature_context (distribution)
  • check_sensitivity (what-if)

Communicator agent

Receives the investigation report and generates a plain-English explanation via Hugging Face LLM (template fallback if unavailable).

Memory layers

  • Working memory (LangGraph state)
  • Session memory (in-process)
  • Long-term memory (PostgreSQL)

Observability

Every agent run is traced via LangSmith — tool call sequence, latency, and token usage.
